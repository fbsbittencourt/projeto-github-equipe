# Projeto Colaborativo — Instituto Casa do Pai

Este projeto tem como objetivo **exercitar o trabalho colaborativo com Git e GitHub**, por meio de **forks, branches, commits e pull requests**.

---

## Objetivo

Cada aluno deverá:

1. Fazer **fork** deste repositório  
2. Criar uma **branch** com seu nome  
3. Criar um arquivo `nomedoaluno.html`  
4. Adicionar o link da sua página no menu do `index.html`  
5. Fazer **commit** e **push**  
6. Enviar um **Pull Request (PR)** para este repositório

---

Se o aluno se chama João:

1. Criar o arquivo `joao.html`
2. Adicionar no `index.html`:
